💰 Finova — Finance Tracker App
Track your money smarter.
A modern, clean mobile finance application UI designed in Figma — built as part of the MainCraft Internship Program.

📱 App Overview
Finova is a personal finance management app designed to help users track spending, manage multiple bank accounts, log expenses, and visualize their financial habits — all in one place.
The prototype covers a complete user journey across 7 app screens:
ScreenDescriptionSplash ScreenOnboarding / Brand entry pointLoginEmail & password authentication with Google sign-inDashboardTotal balance, monthly overview, recent transactionsAdd ExpenseExpense logging with category, date & notesProfileUser settings, bank accounts, preferences & logoutMy WalletMulti-account wallet with quick actionsAnalyticsSpending by category (donut chart) + bar chart overview

✨ Features Designed
🔐 Login & authentication flow
🏠 Home dashboard with balance summary
📊 Monthly spending overview with line chart
➕ Add expense with categories (Food, Health, Travel, Bills)
👤 User profile with preferences & dark mode toggle
🏦 Multi-bank wallet (HDFC, ICICI, Paytm)
📈 Analytics with spending breakdown by category
